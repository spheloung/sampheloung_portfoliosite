# Image Folder Structure

This folder contains all images for the portfolio website, organized by page/section.

## Directory Structure

```
images/
├── hero/                  # Hero section images
├── about/                 # About page images
├── career/                # Career page images
├── projects/              # Project images
│   ├── visa-tracker/
│   ├── billing-engine/
│   ├── portfolio-website/
│   └── rise-of-china/
└── photography/           # Photography location images
    ├── antarctica/
    ├── tekapo/
    ├── canterbury/
    └── coromandel/
```

## Image Guidelines

- **Format**: Use WebP for best compression, with JPEG/PNG fallbacks if needed
- **Sizing**: Optimize images before adding (use Squoosh.app or similar)
- **Naming**: Use lowercase with hyphens (e.g., `hero-main.webp`, `thumbnail.jpg`)

## Recommended Dimensions

| Usage | Recommended Size |
|-------|-----------------|
| Hero backgrounds | 1920x1080 or 2560x1440 |
| Project thumbnails | 800x600 |
| Photography gallery | 1200x800 (or original aspect ratio) |
| Profile/About photos | 600x600 or 800x800 |
